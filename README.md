# COVID-19: The Hidden Death Toll
An interactive Power BI dashboard analyzing estimated cumulative excess deaths during COVID-19 across 122 countries (2020-2026), built using data from Our World in Data.
## Dashboard Pages
Page 1 - The Hidden Death Toll
- KPI Cards (Confirmed Deaths, Lower Bound, Central Estimate)
- Line Chart - Excess Deaths Over Time (2020-2026)
- Donut Chart - Confirmed Deaths by Central Estimate
- Bar Chart - Top Countries by Excess Deaths
- Area Chart - Uncertainty Range
- Stacked Column - Central Estimate by Year and Entity
- Slicer - Entity filter
Page 2 - Waves of Loss
- Table - Country-wise Death Toll
- Waterfall Chart - Rising Toll - Quarter-wise surge
- Treemap - Faces of the Pandemic
- Gauge Chart - Mortality Gauge-577.09K
- Pie Chart - Death Toll by Year (2020-2025)
- Area Chart - The Uncertainty of Loss
- Slicers - Code & Year filters
  
Data Source
Our World in Data-Excess Mortality Dataset
https://ourworldindata.org/excess-mortality-covid
Records: 577,090 rows | Countries: 122 | Period: 2020–2026

Tools Used
Power BI Desktop | Microsoft Excel

Key Insights
Global confirmed COVID deaths: 43 Million
Excess deaths >> Confirmed deaths — hidden toll revealed
Q3 & Q4 highest death surges globally
